# Gots

Call remote Go methods directly from Javascript through WebSocket.

See `examples` folder.

# Helloword

```shell
# cd ANY_GO_MODULE_DIR
go run github.com/discoverkl/gots-examples/helloworld
```

**OR**

```shell
git clone https://github.com/discoverkl/gots-examples.git
cd helloworld
go run .
```
